## Documentation Principles

1. **Scientific Rigor:** All claims based on empirical data
2. **IP Protection:** Do not reveal HVM implementation details
3. **Clarity:** Accessible to AI researchers without prior HVM knowledge
4. **Completeness:** Cover all aspects of the discovery
5. **Professionalism:** Academic format, appropriate citations
